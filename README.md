# Weather API Fetcher

## Description
A weather application that fetches real-time weather data using OpenWeatherMap API.

## Features

- Search weather by city,district
- Current location weather
- Temperature display
- Humidity display
- Wind speed display
- Weather icons
- Responsive design
- User-friendly interface

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## How To Run

1. Download project
2. Open in VS Code
3. Add your API key in script.js
4. Open index.html with Live Server
